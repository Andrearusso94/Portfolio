# Portfolio — Frontend Developer

Portfolio single-page (React 19 + TypeScript + Vite + Vitest), estetica gotham/noir blu.
Sviluppo per issue GitHub tramite orchestratore autonomo con review umana obbligatoria.

Leggi **CLAUDE.md** per architettura, comandi e regole dell'agente.

```bash
npm ci
npm run dev        # dev server
npm run typecheck  # tsc --noEmit
npm test           # smoke test baseline
npm run build      # typecheck + build produzione
```
